## O que o Papiflix faz?

- Ativa um **modo teatro/cinema** limpo e imersivo no DLive
- Esconde elementos desnecessários da interface
- Aumenta o player de vídeo para aproveitar melhor a tela
- Mantém a experiência fluida para assistir filmes, animes, séries ou qualquer conteúdo que o streamer esteja passando
- Leve, sem anúncios extras e sem coletar dados
  
## Recursos planejados / já implementados

- [x] Modo teatro
- [x] Esconder barra superior / elementos distrativos
- [x] Player em tela cheia inteligente (mantendo proporção)

## Como instalar
### Opção 1 – Chrome / Edge / Brave (recomendado)

1. Baixe o .zip
2. Descompacte a pasta
3. Vá em `chrome://extensions/`(funciona em qualquer browser chromium)
4. Ative o **Modo desenvolvedor** (canto superior direito)
5. Clique em **Carregar sem compactação** e selecione a pasta descompactada
6. Pronto! O ícone do Papiflix aparece na barra de extensões.
